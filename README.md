Twitter_haml
================

## Objectives

Re-create the twitter front end using SCSS & HAML

## Technologies
* SCSS
* Haml
* Sinatra
* Flexbox

## How to run
```sh
git clone https://github.com/csharpd/twitter_haml.git
cd sass_twitter
bundle
rackup
```

Open your browser and go to [localhost:9292](http://localhost:9292)
